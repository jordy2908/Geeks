## Installation
```git
git clone https://github.com/jordy2908/Geeks.git
```

## Usage

```bash
cd geeks
```
```laravel
composer install
```

## .env
Change database environment variables (postgresql)
```laravel
DB_CONNECTION=pgsql
DB_HOST=127.0.0.1
DB_PORT=5432
DB_DATABASE=geeks
DB_USERNAME=db_username
DB_PASSWORD=password
```


# Run
```laravel
php artisan key:generate
php artisan migrate
php artisan serve
```
